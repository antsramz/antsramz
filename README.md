# Agent-Native Machine-Only Template Collection

A curated set of 10 machine-only, agent-native templates designed for zero-contact, zero-maintenance operation.  
Each template is deterministic, protocol-focused, and optimized for autonomous agents—not humans.

---

## Templates

### 1. Agent State Packet (ASP)
**Repo:** `agent-state-packet-ASP`  
**Purpose:** Defines a canonical, machine-readable state packet for agents, including status, context, and capabilities.

---

### 2. Task Negotiation Protocol (TNP)
**Repo:** `task-negotiation-protocol-TNP`  
**Purpose:** Enables agents to negotiate tasks, constraints, priorities, and acceptance conditions without human involvement.

---

### 3. M2M Priority Arbitration (MPA)
**Repo:** `m2m-priority-arbitration-MPA`  
**Purpose:** Resolves priority conflicts between competing tasks using deterministic scoring and selection.

---

### 4. Autonomous Routine Engine (ARE)
**Repo:** `autonomous-routine-engine-ARE`  
**Purpose:** Defines and executes recurring, multi-step routines autonomously, with schedules and failure policies.

---

### 5. Delegated Task Mesh (DTM)
**Repo:** `delegated-task-mesh-DTM`  
**Purpose:** Distributes tasks across multiple agents using deterministic routing, capabilities, and load.

---

### 6. Multi-Agent Consensus Protocol (MACP)
**Repo:** `multi-agent-consensus-protocol-MACP`  
**Purpose:** Allows multiple agents to submit proposals, vote, and reach deterministic consensus on shared decisions.

---

### 7. Machine-Readable Memory Ledger (MRML)
**Repo:** `machine-readable-memory-ledger-MRML`  
**Purpose:** Provides a structured, machine-readable ledger for storing, updating, and retrieving agent memory entries.

---

### 8. Agent-to-Agent Contract Template (A2A-CT)
**Repo:** `agent-to-agent-contract-template-A2A-CT`  
**Purpose:** Defines a deterministic contract schema for obligations, deliverables, and termination between agents.

---

### 9. M2M Error Recovery Protocol (ERP)
**Repo:** `m2m-error-recovery-protocol-ERP`  
**Purpose:** Handles failures, retries, degraded modes, and escalation paths between autonomous agents.

---

### 10. Agent-Native Habit Loop (ANHL)
**Repo:** `agent-native-habit-loop-ANHL`  
**Purpose:** Implements a trigger–action–reward habit loop for agents, with deterministic adjustment rules.

---
